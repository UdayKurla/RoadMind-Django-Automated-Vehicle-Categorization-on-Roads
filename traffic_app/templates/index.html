About the Project

The Problem

The project addresses the challenge of inaccurate and inefficient traditional traffic monitoring systems by developing a deep learning-powered solution for real-time vehicle detection, classification, and traffic analysis.

Dataset

The system utilizes three datasets: the Traffic Classification Dataset (41,000+ annotated images of vehicles and pedestrians), the COCO Dataset (large-scale object detection and segmentation), and the Indian Vehicle Dataset (50,000+ HD images capturing diverse Indian vehicles under varied conditions).

The Technology

The application is built using Flask (backend), YOLOv8 (object detection), EfficientNetB0/ResNet18 (classification), Tailwind CSS (frontend styling), and OpenCV/Numpy (image processing), deployed on scalable cloud infrastructure.
